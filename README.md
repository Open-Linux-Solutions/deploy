# Deploy
Custom image based on Debian, to contain all the resources to execute a good pipeline of your projects.

### Resources 

* PHP 7
* NodeJS 14.x
* Yarn
* Composer
* WP CLI

### Installed

* Git
* rsync
* zip and unzip
* MySQl Client
* python-pip
